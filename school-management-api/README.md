# 🎓 School API (Laravel 13)

A clean REST API project built using Laravel following repository pattern architecture and Swagger documentation.

---

## 📌 Project Overview

This project manages student records using a clean architecture approach:

- Controller handles HTTP requests
- Repository handles database logic
- Model represents database structure
- Swagger documents all APIs

---

## 🧠 Core Concept

This project is designed for:

✔ Clean code separation  
✔ Scalable backend structure  
✔ Easy testing and maintenance  

---

## ⚙️ Tech Stack

- Laravel 13
- PHP 8.2+
- MySQL
- L5-Swagger
- Repository Pattern
